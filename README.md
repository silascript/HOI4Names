# HOI4Names
#### 这是个多个mod集合包，主要是各类命名相关的小mod

#### 此mod集合包主要针对中华民国
### CHIForceNames
#### 各类舰船等名字

### CHIDivisionsNames
#### 部队史实名系统
###### 由52PCGame网友**kenlam1980**提供

#### 舰船命令规则如下:
* 驱逐舰: 		非人口Top25及省会或城市名
* 轻巡洋舰:		著名关卡、著名湖泊
* 重巡洋舰:		人口排名Top25及名省省会城市的城市名
* 战列舰:		大江大河及名山
* 战列巡洋舰:		古代名将
* 航空母舰:		省份名
* 潜艇:			古代名剑
